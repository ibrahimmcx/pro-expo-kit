# Workflow: Learn Solution

Use this when the user wants to manually save a snippet or a specific pattern as a "Knowledge Item".

## 1. Capture
- Identify the target code or config.
- Ask the user for a brief description of "why" this is useful.

## 2. Formatting
- Activate `archivist` agent.
- Create a new markdown file in `.agent/knowledge/` following the standard structure.

## 3. Confirmation
- Report back with: "Usta, bilgiyi sandığa attım! Bir sonraki sefere buradan bakacağım: [filename]".
